About

Investment tracker is a platform that allows you to track aspects that affect your investments. Investments that share the same risks, factors, regions and groups can be visually distinguished through the generated relationships graphs for each aspect. Currently you can add investments, factors, risks, regions and group them.

Written in C# using ASP.Net MVC 5 using Entity Framework 6 and using MSSQL database for data persistance and D3 visualization javascript API.
Bootstrap and jQuery used in the front end.

This app is primarily meant to be used to log your investments, define them and understand the affects that factors have on your investments.

Requirements

  1. Place to keep track of all my investments
  2. See relationship between investments
  3. Predict the value of investments based on the relationships to known outcomes
  4. Track the history of an investment
